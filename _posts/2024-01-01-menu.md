### 醋泡花生米

|  主料  |
| :----: | :---: |
| 花生米 | 250克 |

|  料汁  |
| :----: | :---: |
|  陈醋  | 150克 |
|  香醋  | 60克  |
|  白糖  | 30克  |
|  生抽  | 50克  |

|  香料  |
| :----: | :---: |
|  八角  |  5克  |
|  香叶  |  2克  |
| 干辣椒 |  3克  |
|  桂皮  |  1克  |

|  辅料  |
| :----: | :---: |
| 胡萝卜 |  1根  |
|  芹菜  |  1根  |
| 小米辣 | 适量  |
|  大蒜  | 适量  |

制作过程：  
1、花生米冷水下锅，加入香料和盐  
2、水烧开捞出，加入煮花生米的汤浸泡放凉  
3、热锅，加入两种醋，小火烧开，再加入姜片、生抽和白糖搅拌均匀，盛出后放凉  
4、胡萝卜、芹菜斜刀切段，芹菜焯水，大蒜切片  
5、花生米沥干水分，加入辅料，倒入醋汁，浸泡1小时以上

### 包菜炒粉丝

| 主料  |
| :---: | :---: |
| 包菜  | 适量  |
| 粉丝  | 适量  |
| 鸡蛋  |  2个  |

|  料头  |
| :----: | :---: |
|  大蒜  | 适量  |
| 小米辣 | 适量  |

制作过程：  
1、包菜去芯后切丝，粉丝泡软后剪断，与少许老抽、油混合均匀  
2、大蒜切末，小米辣切段，鸡蛋打散后加入少许清水混合均匀  
3、锅中加入适量油，烧热后倒入鸡蛋，炒散后盛出  
4、锅中加入适量油，烧热后加入料头，爆香后加入包菜丝，大火炒至断生  
5、加入少许盐、白糖和生抽调味，放入鸡蛋和粉丝，翻炒至粉丝变干  
6、淋入少许锅边醋，撒上葱花

### 干煸四季豆

|   主料   |
| :------: | :---: |
|  四季豆  | 400克 |
|  五花肉  | 150克 |

|   芽菜   |  5克  |
|   大蒜   |  3瓣  |
|    姜    |  3克  |

制作过程：  
1、四季豆掐去头尾对半折断，五花肉去皮切成肉末，姜蒜切末

3、加入少许盐，将四季豆煸至表面发皱后盛出  
4、锅中加入少许油，将肉末煸至变色后加入花椒、辣椒、姜蒜末、芽菜  
5、翻炒至辣椒变色后加入四季豆，翻炒均匀后加入盐、少许白糖，淋入锅边醋

### 麻婆豆腐

|     主料     |
| :----------: | :---: |
|     豆腐     |  1块  |
| 牛肉（稍肥） | 50克  |

|  料头  |
| :----: | :---: |
|  泡姜  | 15克  |
|  泡椒  | 20克  |
|  蒜苗  |  1根  |

|  调料  |
| :----: | :---: |
| 刀口辣椒 | 5克 |
| 豆瓣酱 | 适量 |
|  豆豉  | 适量 |

制作过程：  
1、豆腐切1.2厘米的块，牛肉剁末，泡姜、泡椒切末，蒜苗切碎  
2、锅中加入适量清水和少许盐，将豆腐块下锅，煮至水开后捞出  
3、锅中加入适量油，烧热后再加入适量猪油烧化，将牛肉末下锅，中小火炒干爆香  
4、加入豆瓣酱、豆豉、泡姜和泡椒，小火炒出红油，再加入刀口辣椒翻炒均匀  
5、加入适量清水，大火烧开，再加入少许盐、白糖和老抽调色  
6、加入豆腐，小火将豆腐烧入味  
7、加入适量水淀粉勾薄芡，搅拌均匀后再次勾芡，最后勾浓芡，淋入少许明油后出锅，撒上适量刀口辣椒和蒜苗碎

P.S.  
1、豆腐不限种类，老豆腐比较香，嫩豆腐口感好  
2、三次芡需要一次比一次浓且一次比一次少

### 洋葱炒鸡蛋

|  主料  |
| :----: | :---: |
|  洋葱  | 300克 |
|  鸡蛋  |  3个  |

|  辅料  |
| :----: | :---: |
|  青椒  | 半个  |
|  红椒  | 半个  |

| 调料 |
| :---: | :---: |
|  盐  |  2克  |
| 白糖 |  2克  |
| 白醋 | 0.5克 |
| 蚝油 |  4克  |

制作过程：  
1、洋葱去除头尾后对半切开，切成小块，青椒、红椒切成小块，鸡蛋稍微搅散  
2、锅中加入适量油，烧至5成热，加入洋葱和青红椒，中火炒至断生后盛出  
3、锅中重新加入适量油，烧至5成热，离火，倒入蛋液煎至定型

| :---: | :---: |

## 荤菜

### 菠萝咕咾肉

|  主料  |
| :----: | :---: |
| 五花肉 | 300克 |

|   料汁   |
| :------: | :---: |
| 番茄沙司 | 50克  |
|    盐    |  1克  |
|   白糖   | 20克  |
|   白醋   | 15克  |

|  辅料  |
| :----: | :---: |
|  青椒  | 半个  |
|  红椒  | 半个  |
| 胡萝卜 | 半根  |
|  黄瓜  | 半根  |
|  菠萝  | 适量  |

制作过程：  
1、五花肉去皮后切片，青椒、红椒、菠萝切块，胡萝卜、黄瓜切滚刀块  
2、料汁混合均匀  
3、肉片加入姜末、葱花、盐、白胡椒粉、料酒、香油，搅拌均匀腌制10分钟  
4、肉片中打入1个鸡蛋，搅拌均匀后加入玉米淀粉，搅拌均匀后将肉片团成球状  
5、锅中加入适量油，烧至5成热，将肉团下锅炸至定型后捞出控油  
6、将胡萝卜炸熟后捞出控油  
7、锅中加入适量油，烧至7成热，将肉团下锅炸至金黄后捞出控油，黄瓜淋油  
8、锅中加入适量油，烧热后下入蒜蓉和葱花爆香，倒入料汁搅拌均匀，另起一锅，将适量油烧热  
9、料汁烧开后勾浓芡，淋入热油，搅拌均匀  
10、加入辅料，翻炒均匀

### 葱烧海参

|   主料   |
| :------: | :---: |
| 水发海参 | 一斤  |
|   大葱   |  3根  |
|  上海青  |  2棵  |
|   高汤   | 适量  |

制作过程：  
1、海参剖开去除内脏后洗净  
2、冷水下锅，煮出异味后捞出冲洗干净  
3、葱白切1寸半段，两侧改斜一字花刀，葱叶切葱条  
4、锅中加入适量油，烧至5成热，小火将葱段炸至微黄后捞出  
5、留少许葱油，加入葱条小火炒至微黄，盛出部分葱条  
6、加入高汤和海参烧开，放入黄酒、盐、胡椒粉和姜片，小火煨20分钟后捞出海参  
7、锅中加入少许葱油，下葱姜煸香  
8、加入酱油、高汤和海参烧开，再加入黄酒、白糖和胡椒粉翻炒均匀，最后加入葱段  
9、收汁后捞出摆盘，上海青焯水后摆盘  
10、剩余汤汁中加入适量高汤、料酒、老抽和水淀粉，混合均匀后过筛，加入少许葱油，混合均匀后淋在海参上

P.S.  
1、可以用炸完的葱加水烧开代替高汤

| :----: | :---: |

| :--------------: | :---: |

| :----: | :---: |

|  花椒  |  3克  |
|  八角  |  1个  |
| 小茴香 |  3克  |
|  桂皮  |  1克  |

|        调料        |
| :----------------: | :---: |
|       豆腐乳       |  1块  |

|       豆瓣酱       | 10克  |
|        红油        | 适量  |

|   辅料    |
| :-------: | :---: |
| 红薯\土豆 |  1块  |

制作过程：

2、热锅，加入大米和糯米，中火炒至水分蒸发

6、牛肉分次加入米粉，翻拌均匀，加入红油抓拌均匀  
7、辅料切薄片垫底，码上牛肉  
8、开水上笼，中小火蒸50分钟，表面撒上葱花

### 锅包肉

|  主料  |
| :----: | :---: |
| 里脊肉 | 200克 |

|   挂糊   |
| :------: | :---: |
| 土豆淀粉 | 100克 |
|    油    | 10克  |

|  辅料  |
| :----: | :---: |
| 胡萝卜 | 10克  |
|  香菜  | 10克  |
|  大葱  |  5克  |
|  生姜  |  5克  |
|   蒜   |  5克  |

| 料汁  |
| :---: | :---: |
| 白糖  | 30克  |
| 白醋  | 20克  |
|  盐   |  3克  |
| 清水  | 适量  |

制作过程：  
1、里脊肉切3毫米的薄片，加入适量盐、胡椒粉、料酒，抓拌均匀  
2、土豆淀粉加入适量清水泡开，静置30分钟，倒出上层清水，取出湿淀粉  
3、胡萝卜、香菜、大葱、生姜切成细丝，大蒜切片  
4、肉片加入湿淀粉和油抓拌均匀  
5、锅中加入适量油，烧至5成热，将上好浆的肉片依次下入锅中，中小火炸至定型后捞出  
6、锅中加入适量油，烧至7成热，将肉片下锅复炸30秒后捞出  
7、将料汁下锅烧开，熬至黏稠，加入辅料翻炒均匀  
8、将炸好的肉片下锅，将汁均匀裹在肉片上

4、转中小火加入料头，炸至金黄色后捞出料渣  
5、油回温至5成热后加入糍粑辣椒，中小火翻炒至略干  
6、加入干辣椒段，翻炒均匀后加入豆瓣酱，小火炒香  
7、加入姜末，豆豉，小火炒出香味后加入香料炒5分钟  
8、加入花椒，3分钟后加入冰糖炒化  
9、加入醪糟炒2～3分钟，再加入白酒，熬5分钟后关火静置24小时

#### 水发海参

|   主料   |
| :------: | :---: |
| 淡干海参 | 50克  |

制作过程：  
1、海参洗净，蒸50分钟后取出  
2、冷水下锅，烧开后煮5分钟，关火放凉  
3、去除海参肠和海参牙，洗净后放入清水中，冷藏24小时  
4、冷水下锅，烧开后煮1分钟，捞入另一冷水锅，再次烧开后放入冷水中冷藏24小时  
5、重复两至三次

P.S.  
1、泡发过程中全程不能沾油

#### 水炼猪油

|  主料  |
| :----: | :---: |
| 猪板油 |  1块  |

制作过程：  
1、猪板油切丁  
2、冷水下锅，淋入少许白酒，水微开后捞出洗净  
3、锅中加入适量水，烧至微开，加入猪油，转小火，加入适量姜片、花椒  
4、不停翻动，蒸发水分  
5、稍微出油时淋入少许白醋  
6、充分出油后捞出油渣，放凉

#### 水洗面筋

|   主料   |
| :------: | :-------: |
| 高筋面粉 |   一斤    |
|   清水   | 240克左右 |

制作过程：  
1、面粉分次加入清水搓拌成面絮  
2、面团揣匀揣光后醒面10分钟  
3、盆中加凉水没过面团，把面团不断往中间揉压，洗到面浆浓白后，过滤出面浆  
4、重复2～3遍后用两个手轻轻拢住不停翻个，轻轻揉压，直到清洗不出面浆

#### 糖色

| 主料  |
| :---: | :---: |
| 冰糖  | 适量  |

制作过程：  
1、冰糖敲碎，倒入等量的清水，再加入适量植物油  
2、中小火倒入糖水，烧开，转小火，不断搅拌至水分蒸发  
3、水分蒸发后气泡变小，可制作挂霜菜品  
4、糖浆搅拌至香油色后，可制作拔丝菜品  
5、糖浆搅拌至黄色变深后 ，可制作冰糖葫芦  
6、糖浆变成浅枣红色后关火，此时加入开水后为嫩汁  
7、利用余温将糖浆搅拌至深枣红色后加入两倍的开水  
8、重新开火熬煮1分钟

#### 炸蛋

| 主料  |
| :---: | :---: |
| 鸡蛋  |  3个  |

制作过程：  
1、鸡蛋打散，加入少许盐搅拌均匀

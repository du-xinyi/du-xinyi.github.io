# docker相关笔记
## docker ros可视化
以`osrf`中的`ros:humble-desktop`为例。
在容器外的终端执行如下命令，允许任意主机连接到本机的X server
```shell
xhost +
```
随后启动容器
```shell
sudo docker run -it -v /tmp/.X11-unix:/tmp/.X11-unix -e \
DISPLAY=$DISPLAY osrf/ros:humble-desktop bash
```
即可使用`rviz2`等进行可视化显示
